## 提案5パターン

### ① Luxury Mono（ラグジュアリー・モノトーン）
- **コンセプト**: 漆黒 × シャンパンゴールドで"格"を出す、ハイエンドBtoBの静謐な佇まい
- **カラー**: `#000` / `#FFF` / `#8C7853`（ゴールド）
- **フォント**: Shippori Mincho + Cormorant Garamond + Noto Sans JP Light
- **推奨度**: ★★★☆☆（競合との差別化に振り切りたい場合に有効）
- **特徴**: 余白を最大化した縦長ヒーロー、モノクロ写真、極細ウェイト一貫
- **リスク**: 情報量の少なさがBtoB顧客の「もっと知りたい」要求と衝突しやすい

### ② Clean Medical（クリーン・信頼型）★ 第一推奨
- **コンセプト**: 医療のような"清潔感＋信頼感"で、ネットワークの主治医としてのポジショニング確立
- **カラー**: `#2B4C7E`（信頼ブルー）+ `#5A7F6F`（セージ）+ ベージュ
- **フォント**: Plus Jakarta Sans + Noto Sans JP
- **推奨度**: ★★★★★
- **特徴**: 不→満の可視化ブロック、数値バンド（KPI）、4カード構成サービス、問合せ導線を随所に
- **理由**: 後述

### ③ Wa-Modern（和モダン・力強い型）
- **コンセプト**: 「道」「志」といった漢字一文字を主役に、情熱と誇りで差別化
- **カラー**: `#0F1020`（黒紺）+ `#D4AF37`（金）+ `#C1272D`（朱）
- **フォント**: Shippori Mincho Bold + Cormorant Garamond
- **推奨度**: ★★★☆☆
- **特徴**: ヒーローで「道」の巨大漢字、三不三満のダーク背景グリッド、セリフ引用ブロック
- **リスク**: 堅実なBtoB顧客が期待する"安心感"より"挑戦感"が先に立つ

### ④ Craft Warm（クラフト・温かみ型）
- **コンセプト**: ブラウン＋キャラメルで「手仕事のような通信」を演出
- **カラー**: `#3D2817`（ダークブラウン）+ `#B45309`（キャラメル）+ `#F4EFE8`（生成り）
- **フォント**: DM Serif Display + Noto Serif JP
- **推奨度**: ★★★☆☆
- **特徴**: スタンプ風バッジ、4ステップの工程、温かい職人像の画像使用
- **リスク**: 通信インフラの"硬派さ"とのギャップが好悪分かれる

### ⑤ Artistic Void（アート・余白型）
- **コンセプト**: 「間」「静」「光」の単漢字を大胆に配置、余白で語るミニマル
- **カラー**: `#F5F0E8`（和紙）+ `#1A1A1A`（墨）+ `#8B0000`（朱）
- **フォント**: Shippori Mincho Light
- **推奨度**: ★★☆☆☆
- **特徴**: ハンバーガーナビだけの最小UI、非対称グリッド、抽象的なマニフェスト
- **リスク**: BtoB顧客の「分かりやすさ」「情報の透明性」への要求と方向が逆

## アオからの第一推奨

### 第一推奨: ② **Clean Medical（クリーン・信頼型）**

#### 選定理由（200字）

ロードの顧客は、通信トラブルで業務が止まる痛みを嫌う法人です。訴求の軸は"派手さ"ではなく"安心"。Clean Medical は医療のような清潔感・白基調・ブルーアクセントにより「任せれば大丈夫」を即座に伝えます。さらに"3つの不→3つの満"の可視化ブロックと数値バンド（初動3分・コスト20%減）を配置でき、理屈と感情の両面でBtoBの意思決定を支援。現行サイトのシンプルさを継承しつつ、CVR向上に直結する構造です。

#### 第二候補: ① Luxury Mono

もしロードが「上流の大企業・プレミアム案件」に振り切りたい場合は Luxury Mono を推奨。ただしリード獲得より"格"を優先する設計のため、中小企業向けの問合せ量は減るリスクがあります。
