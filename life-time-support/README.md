## 提案5パターン

### ① Luxury Mono（ラグジュアリー・モノトーン）

- **コンセプト**: 「引き算の美学」。漆黒と純白のみで構成し、新居という人生の節目に高級感の付加価値を纏わせる。
- **推奨度**: ★★★☆☆
- **カラー**: `#000000` / `#FFFFFF` / アクセント: シャンパンゴールド `#8C7853`
- **フォント**: Noto Serif JP（和文見出し）/ Cormorant Garamond（欧文見出し）/ Inter Light（本文）
- **適合理由**: ハイエンドマンション購入層には刺さるが、LTSの親しみやすい現行トーンからは飛躍が大きい
- **ファイル**: `01-luxury-mono.html`

### ② Clean Medical（クリーン・信頼型）

- **コンセプト**: 清潔感・分かりやすさ・信頼感を全面に。現行サイトの延長線上で正当進化。
- **推奨度**: ★★★★☆（次点推奨）
- **カラー**: ベース白 `#FFFFFF` / 信頼ブルー `#2B4C7E` / 暖色アクセント `#F4EFE8`
- **フォント**: Noto Sans JP（和文）/ Inter（欧文）
- **適合理由**: 現行の白・グレー・濃紺基調から自然な進化。CV導線と情報構造が明瞭で、問い合わせ率向上に直結しやすい
- **ファイル**: `02-clean-medical.html`

### ③ Wa-Modern（和モダン・力強い型）

- **コンセプト**: 深紺と金で「志」と「品格」を演出。日本の伝統美と現代のモダンさを融合させ、プレミアム路線へ。
- **推奨度**: ★★★☆☆
- **カラー**: 深黒紺 `#1A1A2E` / 金 `#D4AF37` / 和紙色 `#F5EFE0` / 朱 `#C1272D`
- **フォント**: Shippori Mincho（和文見出し）/ Cormorant Garamond（欧文）
- **適合理由**: 高価格帯・こだわり派顧客への訴求は強いが、ファミリー層への親しみには少し距離感
- **ファイル**: `03-wa-modern.html`

### ④ Craft Warm（クラフト・温かみ型） ★第一推奨

- **コンセプト**: 手仕事の温もりと、ご家族の新生活に寄り添う情緒価値。素材感を全面に出し、共感でCVへ導く。
- **推奨度**: ★★★★★
- **カラー**: ダークブラウン `#3D2817` / 生成り `#F4EFE8` / 琥珀 `#B45309`
- **フォント**: Noto Serif JP（和文）/ DM Serif Display（欧文）/ Inter（本文）
- **適合理由**:
  1. 「新居での暮らし」の情緒価値を最も表現できる
  2. エコカラット・コーティング・家具など **質感を持つ商材** と温かみのビジュアルが高親和
  3. ファミリー層に自然に共感を生む
  4. 越谷という地域密着の「血の通った事業者感」を出せる
  5. 現行サイトから大きく飛躍しつつも、親しみやすさの核は維持
- **ファイル**: `04-craft-warm.html`

### ⑤ Artistic Void（アート・余白型）

- **コンセプト**: 「間（ま）」の美学。余白と書体で暮らしの哲学を語る、ギャラリー的アプローチ。
- **推奨度**: ★★☆☆☆
- **カラー**: 和紙色 `#F5F0E8` / 墨 `#1A1A1A` / 朱 `#8B0000`
- **フォント**: Shippori Mincho / Cormorant Garamond
- **適合理由**: 個人ブランド・アーティスト向けの性格が強く、BtoCマスマーケットには情報量不足
- **ファイル**: `05-artistic-void.html`

## アオからのおすすめ

**第一推奨: ④ Craft Warm**

新築マンション購入というライフイベントは、購入者にとって **「理性（品質・価格）」と「感情（わくわく・憧れ）」の両輪** で意思決定される。御社の事業特性（素材・手仕事・伴走）は後者に強い説得力を持ちつつ、「高品質・適正価格」という前者も同時に訴求できる稀有なポジショニング。

Craft Warm は、この両輪を**温度感のあるビジュアル**と**明快な情報設計**で同時に可視化するパターン。ご家族の新しい生活と御社の手仕事を地続きに見せられる点で、他の4パターンを上回ると判断しました。

**次点: ② Clean Medical**

即効性と現行ブランドからの親和性を重視するなら Clean Medical。リニューアル負担を抑えつつCVRを底上げする堅実案です。

## 技術メモ

- **HTML**: 全パターン単体ファイル（外部依存は Google Fonts のみ）
- **CSS**: 共通リセットは `assets/common.css`、パターン固有は `<style>` 内
- **画像**: Unsplash Source API による仮素材。**本番では差し替え必須**（撮影推奨カット: ショールーム全景 / 施工例 before-after / スタッフポートレート / 越谷の街並み / 無料内覧会同行の様子）
- **レスポンシブ**: 900px 以下でモバイル対応
- **アクセシビリティ**: 文字コントラスト WCAG AA 目安で設計、`alt` 属性付与
- **Core Web Vitals**: 本番実装時は画像最適化・フォント `display:swap` の徹底を推奨
