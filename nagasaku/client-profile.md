## 事業内容

茨城県鹿嶋市を拠点とする**食品卸問屋**。
食品スーパーへ向けて、以下の食品カテゴリを安定供給している。

### 主要取扱商品
1. 醤油・味噌などの調味料
2. カップ麺・袋麺
3. 缶詰
4. 飲料
5. その他乾物・常温食品

### ビジネスモデル
**BtoB**（食品スーパー向け卸売）が主軸。仕入れ → 倉庫保管 → 配送までを一貫運営。

### 取引先
地域の食品スーパー（茨城県鹿嶋市・神栖市周辺を中心とする）

## 沿革のハイライト

- **1945年**: 初代・永作栄之が「永作商店」として佃煮屋を創業。
  漁師から仕入れた魚を東京築地まで自転車で行商（片道30km、未舗装）。
- **2代目**: 父の代で食品卸業へ転換。
- **令和3年12月**: 3代目・永作浩二が事業承継。

→ **「3代続く老舗」というストーリーがブランドの核**。

---

## 差別化軸・強み

1. **80年の業歴**（1945年創業）= 信頼・継続・地域密着
2. **3代続く家業承継** = 地元雇用・誠実な経営
3. **安心・安全な食品供給** = 地域食品インフラとしての責任感
4. **社員・パート従業員の家族を物心両面で幸せにする** = 人を大切にする経営理念
5. **コミュニケーション重視** = 顧客・スタッフ・地域との関係性

## ブランドトーン

- **老舗・伝統**: 80年の重みを生かした落ち着き
- **誠実・実直**: 派手さよりも信頼感
- **地域密着**: 茨城・鹿嶋という土地性をしっかり打ち出す
- **家族的・温かさ**: 「社員とその家族の幸せ」を経営理念に置く温度感
- **食品の安心・安全**: 衛生・品質管理への信頼

→ ラグジュアリー・モノトーンよりも、**和モダン or 温かみのあるクラフト系** が事業特性に合致。
→ ただしBtoBの取引先信頼性を考えると**Clean Medical系（清潔・信頼）も有力候補**。

## 既存サイトの課題（推察）

1. **デザインが古い・WordPressデフォルト感が強い** → ブランド価値が伝わらない
2. **代表挨拶や沿革が断片的** → 80年の歴史的ストーリーが活きていない
3. **取引先（食品スーパー仕入担当者）目線で「取扱商品の幅」「配送エリア」「規模感」が即座に分からない**
4. **採用ページが弱い** → 4tドライバー募集の訴求力不足
5. **モバイル対応が古いテンプレ任せ**
