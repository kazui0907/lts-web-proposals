# LTS WEB Proposals

株式会社ライフタイムサポート（LTSグループ）の **WEBサイト初回提案モック** を公開する専用リポジトリ。

`/hp` コマンド（LTS AI社員システム）によって自動生成されたクライアント提案が、GitHub Pagesで公開される。

## 公開URL

https://kazui0907.github.io/lts-web-proposals/

## 仕組み

- GitHub Pages（main branch / root）で公開
- クライアントごとにサブディレクトリ（例: `/life-time-support/`）
- 各ディレクトリに5パターンのデザイン案 + 比較プレビュー

## 生成元

- リポジトリ: [kazui0907/ai-employee](https://github.com/kazui0907/ai-employee)（private）
- コマンド: `/hp <既存サイトURL>`
- 担当: **アオ**（dept-web-creative）

## 注意

- **提案段階のモック**であり、画像は Unsplash の仮素材
- 本番実装前に写真差替え・コピー精査・法的チェックが必要
- 提案の内容は非公開クライアント情報を含むため、URLの共有範囲は関係者に限定すること
